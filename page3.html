Calculated Columns vs. Measures

Understanding the Core Differences

PART 02

Calculated Columns

Static Extension

Calculation Time

Computed once during data refresh.

Storage Impact

Stored in memory (RAM). Increases model size.

Evaluation Context

Row Context. Calculates row-by-row.

Best Used For

Slicers, filters, axes, and grouping categories.

Example: Static Margin Calculation

GrossMargin =

'Sales'[Amount] - 'Sales'[Cost]

Measures

Dynamic Calculation

Calculation Time

Computed on demand at query time (user interaction).

Storage Impact

CPU intensive. Does not increase file size.

Evaluation Context

Filter Context. Aggregates based on slicers/visuals.

Best Used For

Numerical values, KPIs, percentages, and ratios.

Example: Dynamic Percentage

Gross Margin % =

DIVIDE(

SUM('Sales'[GrossMargin]),

SUM('Sales'[Amount])

)
