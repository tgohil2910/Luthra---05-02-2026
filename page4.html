Basic DAX Functions

Cheat Sheet: Essential Categories

PART 04

Aggregation

Math & Stats
  • SUM Adds numbers in a column
  • AVERAGE Returns the arithmetic mean
  • COUNTROWS Counts the number of rows in a table

Example: Total Sales

Total Sales = SUM('Sales'[Amount])

Order Count = COUNTROWS('Orders')

Logical

Conditionals
  • IF Checks a condition, returns T/F results
  • SWITCH Evaluates an expression against a list of values
  • AND / OR Boolean logic operators (&&, ||)

Example: Status Label

Status = IF([Total Sales] > 100000, "High", "Low")

// Or use SWITCH for multiple cases

Text

String Manipulation
  • CONCATENATE Joins two text strings (or use &)
  • FORMAT Converts value to text with specified format
  • LEFT / RIGHT Returns characters from start/end of string

Example: Formatted Name

Full Name = [First] & " " & [Last]

Label = FORMAT([Amount], "$#,##0")

Date & Time

Time Aware
  • TODAY Returns the current date
  • DATEDIFF Returns count of intervals between dates
  • EOMONTH Returns date of the last day of the month

Example: Days Since Order

Days Ago = DATEDIFF('Sales'[OrderDate], TODAY(), DAY)

End Date = EOMONTH([Date], 0)

Pro Tip: Always use DIVIDE(n, d) instead of / for safe division (handles divide-by-zero).
